Windows NT 4.0

Microsoft Windows NT 4.0 (codenamed Cairo/SUR) is an operating system released in 1996-2001.At first, as a beta 1 version in 1995 and 1997. Then, as a server edition terminal edition service pack 3 edition (codenamed Hydra) and 2 version.Finally In 1996 and 1998 as a terminal server edition service pack 3 edition. This includes robotz, musica, jungle, and utopia sound schemes.
