Deepin

Deepin is a free operating system that uses the Linux kernel.

It is one of the most popular Chinese Linux distributions and it is based on Debian. The goal with deepin is to make it easy to use and install onto a computer. deepin can be used on all types of personal computers. deepin is downloaded as a DVD, which is free to download on the deepin website. It can be installed or tested by running the DVD.

Started in 2004, deepin has been developed at a fast rate by Wuhan Deepin Technology Co., Ltd, a Chinese company.

The release cycle has followed various schedules but currently aims for four releases per year. Releases are delayed if the work of development and testing has not been completed.
deepin is based on Debian and both distributions have a lot in common. Both distributions use the same software repositories. Most differences are on the desktop. Debian has a vastly larger support community than deepin. However, the majority of Debian help and advice is also applicable to deepin. Debian and deepin support also x86-64 CPU architecture.

Many popular multimedia codecs are not installed by default in deepin. deepin and many other gratis GNU/Linux distributions such as Debian and Ubuntu, do not distribute these codecs with the initial install media due to patent encumberment issues.

deepin 2014.x was based on Ubuntu; Ubuntu and deepin both focus on usability, but deepin offers a different user experience (DDE) and it includes a number of applications that are not available in Ubuntu (Deepin Store).
